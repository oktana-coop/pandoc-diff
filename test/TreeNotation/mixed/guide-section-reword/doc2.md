# Getting Started

This guide walks you through installing the toolkit and building your first project on a fresh machine.

## Requirements

Before you begin, make sure you have:

1. A recent version of the runtime installed.
2. At least 4 GB of free disk space for the build cache.
3. Network access to download the starter templates.

When you are ready, running the setup command lets the toolkit prepare a default workspace on your behalf and fetch whatever components are still missing.
