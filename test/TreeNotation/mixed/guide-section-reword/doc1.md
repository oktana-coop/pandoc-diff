# Getting Started

This guide walks you through installing the toolkit and running your first build on a fresh machine.

## Requirements

Before you begin, make sure you have:

1. A recent version of the runtime installed.
2. At least 2 GB of free disk space for the build cache.
3. Network access to download the starter templates.

Once everything is ready, run the setup command and the toolkit will configure a default workspace for you, downloading any missing pieces along the way.
